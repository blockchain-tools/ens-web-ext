# ens-web-ext

web extension for Ethereum Name Service.

bridge between decentrized and traditional www

## Install

	$ yarn

## Development

    yarn dev chrome
    yarn dev firefox
    yarn dev opera
    yarn dev edge

## Build

    yarn build chrome
    yarn build firefox
    yarn build opera
    yarn build edge

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
